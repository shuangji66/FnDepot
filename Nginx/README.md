<p>Nginx是全球最受欢迎的Web服务器、高性能负载均衡器、反向代理、API网关和内容缓存。</p><p>本项目基于官方stable分支，为飞牛os目录结构和应用架构特化，使用docker-debian12编译构建，仅提供x86版本。</p><p><strong>使用方法：</strong></p><p><strong>进入应用文件/Nginx子目录。编辑/etc/nginx.conf添加web服务；/nginx放置静态html资源或cgi动态资源；/lib/modules额外添加nginx模组。重启加载配置。</strong></p><p><strong>编辑nginx.conf时注意路径和用户名。更多使用说明可参考官方文档。</strong></p>
