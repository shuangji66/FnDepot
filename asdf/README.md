<p>asdf 是一个工具版本管理器，可方便管理Ruby、NodeJS、Java、PHP、.Net在内的几百种运行环境，不影响系统的运行环境。</p><p><strong>使用说明：</strong></p><p><strong>安装后打开任意终端，登录你的用户名账户，输入数据目录的环境变量，推荐"export ASDF_DATA_DIR=/var/apps/asdf/home/.asdf"(每次打开终端都需要输入，否则目录默认为~/.asdf)，add添加工具插件仓库并install指定版本，已安装的工具$PATH为"/var/apps/asdf/home/.asdf/installs/插件包名"，也可自定义目录统一管理。更多环境变量和命令参考[官方文档](https://asdf-vm.com/zh-hans/manage/configuration.html)</strong></p>
